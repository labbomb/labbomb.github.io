青春致远

发表于 2017-06-01 | 分类于 随笔 | | 阅读次数

“来也匆匆，去也匆匆，离绪千种，期待着一次重逢……”

——题记

三年前的夏天，我们怀着美好的憧憬走进了一间教室。三年后的这个夏天，我们就要怀着满腔的留恋离开这个学校了。等到下一个夏天，教室里依旧坐满了人，可那些人不再是我们。

三年前，我们不听话，平时打打闹闹，心里甜甜的。喜欢看下课时老师出门的背影，喜欢每次吃完饭一起过泼水节，喜欢一起聊天聊到忘记了时间，喜欢一起为了考进梦想的高中发奋“疯狂”……“老师来了，老师来了，别说话了!”“呸!烂茄子们!”“你看他，你看他，真能耐!”“好‘狗’不挡路!你给我闪开!”“大姐，我下次再也不敢了，放过我吧!”“那谁，你东西掉了……真笨，你被骗了!”还有好多好多话，可惜已经记不清了。那年，我们嘴里叫出来的，是一个一个的外号;那年，我们轻点几步，就当作体育课跑操跑了四圈;那年，我们互相说着经历过的笑话或糗事，几百遍，几千遍，也不觉得厌烦……

三年后，我们坐在教室里，为梦想而拼搏着，但彼此之间的感情依旧老样子，只是，更深了一些。也许，脸上的笑容没以前多了，但笑起来依旧灿烂;也许，聊天的时间没以前久了，但聊起来依旧热火朝天;也许，没时间在一起过属于我们的泼水节了，但过起来依旧疯狂……有时间，我们仍要回忆以前刻骨铭心的记忆，这些美好的回忆，可以作为拼搏的动力。每个人对自己的人生都已作出了选择，该走的老同学还没等到毕业就走了，该留的，还是留下了。

青春这场盛宴，我们走走停停，停停走走，还是快要毕业了。七十多天后，我们人生的第一个转折点——中考和毕业。“毕业就是一窗玻璃，我们注定要撞碎它，然后擦着锋利的碎片走过去，血肉模糊之后开始一个完全不同的人生。”课桌上一直刻着这样一句话。是啊，毕业了，就没有同桌在耳边唠唠叨叨说个没完了;毕业了，就不能撒娇央求别人打水时给自己捎一壶了;毕业了，就再也没有在班里一起听歌的机会了。毕业了，心里有股酸酸的味道……在这个奇妙的夏天，我们终要迎来一场最后的合影。毕业之后，世界这么大，去哪儿的都有。高中三年，上完大学，读完研究生……很多年之后，我们是否会依旧老样子，坐在一起说一说多年的经历，即使到那时，我们身边的朋友已经很多了，但初中三年，毕竟是很值得留恋的过去啊!等我们翻开那些早已泛黄的老照片时，是否还会记得青春里的这三年?

“如何缅怀这场青春盛宴，轻松地说出一句再见，过去的一切系成了死结，将这段回忆用标点省略……”critty的一首《青春不散》已经听了三年，终于到了快要说再见的时候了……曾经，我们抱怨过，感慨过，欢喜过，悲伤过，可如今，我们剩下的是满腔的留恋，满脸的辛酸。曾经，勇敢过，可，还没毕业呢，就不敢面对这突如其来的毕业了，心里，酸酸的……

毕业，就是很酸，饱含了三年的青春，真的很酸很酸。可是，只要青春不老，我们就不散，永远永远……
